# מערכת לימוד – אבטיפוס עם 4 סוכנים

אפליקציה ללימוד מתמטיקה לילדים: **תרגיל + צ'אט עם מורה**.

## ארכיטקטורת סוכנים

| סוכן | תפקיד | מדבר עם התלמיד? |
|------|--------|-----------------|
| **מורה** (Teacher) | הסבר, צ'אט, הנחיה | **כן — זה מה שהתלמיד רואה** |
| **פסיכולוג** | מוטיבציה, עידוד לפי מצב | לא — מייעץ למורה |
| **מתמטיקאי** | בניית תרגילים | לא — מייעץ / יוצר שאלות |
| **מעצב** | איורי SVG (גאומטריה, גרפים) | לא — מציג איור ליד השאלה |

**מפתח API אחד** (Claude או OpenAI) מספיק לכולם.

## איך מדברים עם המורה (הסוכן שלך)

1. התקן [Node.js](https://nodejs.org)
2. העתק `.env.example` → `.env` והדבק מפתח (אחרי שתיצור ב-Anthropic או OpenAI)
3. בטרמינל:

```bash
cd "פרויקט לימוד"
node server.js
```

4. פתח בדפדפן: **http://localhost:8787**
5. כתוב בצ'אט משמאל — **המורה** עונה; השאר עובדים מאחורי הקלעים

> בלי שרת (`node server.js`) ובלי מפתח — עובד **מצב מקומי** (תשובות פשוטות, בלי AI אמיתי).

## מפתח API — כמה צריך?

| שאלה | תשובה |
|------|--------|
| כמה מפתחות? | **אחד** לחשבון Claude או OpenAI |
| מפתח לכל סוכן? | **לא** — אותו מפתח, פרומפטים שונים |
| לתת לי (Cursor) את המפתח? | **לא** — רק בקובץ `.env` אצלך |

### Claude

1. [console.anthropic.com](https://console.anthropic.com) → API Keys → Create
2. ב-`.env`:

```env
LLM_PROVIDER=anthropic
ANTHROPIC_API_KEY=sk-ant-...
```

### OpenAI

```env
LLM_PROVIDER=openai
OPENAI_API_KEY=sk-...
```

## זיכרון מתמשך לסוכנים

לכל סוכן שמתקשר עם התלמיד יש **קובץ JSON אחד** עם רשומה לכל תלמיד (לפי `userId`):

```
data/memory/
  psychologist.json   ← מצב רגשי, ביטחון, מוטיבציה, מה מדרבן, מה מתסכל
  mathematician.json  ← רמה, סטטיסטיקה לכל נושא, מה זורם / קשה / נמנע
  teacher.json        ← איך לדבר, איך להכווין לתשובה, מה עובד, התקדמות
```

- לפני כל תשובה הסוכן **קורא** את הזיכרון שלו ומקבל אותו כהקשר.
- אחרי כל אינטראקציה ה-AI **כותב מחדש** פרופיל מעודכן, והקוד שומר ציר-זמן עובדתי (נכון/טעות/מצב/נושא, עד 60 אירועים אחרונים).
- ה-`userId` נוצר אוטומטית בדפדפן ונשמר ב-`localStorage` — כל תלמיד מקבל זיכרון נפרד.
- בלי מפתח AI הזיכרון לא מתעדכן (אין ניתוח), אבל קבצים קיימים נקראים כרגיל.

## מבנה קבצים

```

index.html
```

## API

- `GET /api/status` — סטטוס AI והסוכנים
- `POST /api/chat` — שיחה עם המורה (+ צוות)
- `POST /api/problem` — תרגיל חדש מהמתמטיקאי

## טיפים בצ'אט

- "תסביר לי" — הסבר מהמורה
- "תרגיל חדש" — המתמטיקאי בונה שאלה
- נושא **גאומטריה** או **פונקציות** — המעצב מוסיף איור
