# מאגר ייחוס לרמות החומר

כאן אפשר להזין דוגמאות שאלות אמיתיות לכל כיתה, כדי שהסוכן המתמטיקאי ילמד את
**הרמה והסגנון** הנכונים כשהוא מייצר שאלות חדשות (שכבת ה‑AI).

## איך מזינים

צרו קובץ בשם `grade-<מספר הכיתה>.md` (או `.txt` / `.json`), למשל:

- `grade-1.md` — כיתה א׳
- `grade-4.md` — כיתה ד׳

בתוך הקובץ כתבו חופשי — כותרת נושא ואחריה כמה שאלות לדוגמה ברמה הנכונה. למשל:

```
## חיבור עד 20
8 + 5 = ?
9 + 9 = ?

## שאלות מילוליות
לדנה 7 תפוחים, היא קיבלה עוד 4. כמה יש לה עכשיו?
```

הסוכן יקרא את הקובץ של הכיתה הרלוונטית וישתמש בו כדי לכוון את הקושי והניסוח של
שאלות חדשות. עד ~1800 תווים ראשונים נכנסים להנחיה.

> נושאים עם מחולל אלגוריתמי (חיבור, חיסור, צורות, שעון, כסף...) ממילא מדויקים —
> חומר הייחוס משפיע בעיקר על נושאים שה‑AI מייצר (שברים, שאלות מילוליות וכו').
